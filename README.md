# Phil Test Repo

A simple Node.js project used for testing the Phil AI coding agent.

## Features

This project provides the following math functions:

- **add(a, b)** - Adds two numbers together
- **subtract(a, b)** - Subtracts the second number from the first
- **multiply(a, b)** - Multiplies two numbers together

## Usage

```bash
npm start
npm test
```

## Contributing

We welcome contributions to this project! Here are some basic guidelines:

1. **Fork the repository** and create a new branch for your feature or bug fix
2. **Write clear, descriptive commit messages** explaining what your changes do
3. **Add tests** for any new functionality you implement
4. **Ensure all tests pass** by running `npm test` before submitting
